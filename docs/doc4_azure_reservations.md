# Azure Reservations Notes

Azure Reservations let customers prepay for compute or other resources for
a 1 or 3 year term in exchange for a discount versus pay-as-you-go rates.

Key points:
- Reservations apply automatically to matching running resources; no code or config change needed on the VM itself.
- Scope can be set to a single subscription, a resource group, or shared across a billing account.
- Unused reservation hours do not roll over or refund by default, though limited exchange/refund policies exist.
- In CSP, reservations are purchased and managed through Partner Center, not the Azure portal.
