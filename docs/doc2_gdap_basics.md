# GDAP Basics

Granular Delegated Admin Privileges (GDAP) replaced the old all-or-nothing
Delegated Admin Privileges (DAP) model for CSP partners.

Key points:
- Access is granted per Azure AD/Entra role, not blanket admin rights.
- Relationships expire automatically after a set duration (max 2 years) and must be renewed.
- Security groups map GDAP roles to specific partner staff, so access can be adjusted without recreating the relationship.
- Customers approve or reject GDAP requests directly in their own tenant admin center.
